wikigraph by emiraga

English Wikipedia as graph

Some of you might remember [Six degrees of Wikipedia](http://www.netsoc.tcd.ie/~mu/wiki/) by Stephen Dolan. That was done in 2007 and I though it would b e a good idead to the same several years later and see what might have changed. I started with a [XML parser](https://github.com/emiraga/wikigraph/blob/f4ee89d28efc93f4b44d7ccea4b036aa3db806f6/xmlparse.py) just like Stephen, but late r decided to parse SQL files instead, since they contain all information we need, and admittedly wiki parsing is much more difficult. As far as database goes, I needed one which was in-memory (including VM capabilities), and one that supports more complex data structures such as queues (distributed proces sing). Needless to say: [**redis**](http://redis.io/) was database of choice.

Dependencies

[hiredis](https://github.com/antirez/hiredis), located in this directory named `hiredis` 13 * [zlib](http://zlib.net/), command `sudo apt-get install zlib1g-dev` in ubuntu.
